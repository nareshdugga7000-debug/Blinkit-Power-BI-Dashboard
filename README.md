# 🛒 Blinkit Sales Analytics Dashboard

## 📊 Project Overview
Developed an interactive Power BI dashboard to analyze Blinkit sales performance, featuring key KPIs such as Total Sales, Average Order Value, and Category-wise insights. Utilized DAX for calculated measures and enabled data-driven decision making through clear visualizations of regional and product trends.

## 🛠️ Tools & Technologies
- Power BI
- Excel
- DAX

## 📌 Key Features
- KPI tracking: Total Sales, Average Order Value
- Category-wise sales analysis
- Region-wise performance insights
- Interactive filters and visuals

## 📈 Key Insights
- Identified top-performing product categories
- Analyzed sales distribution across regions
- Improved decision-making using visual analytics
